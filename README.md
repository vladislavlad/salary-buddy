# Salary Buddy

Калькулятор зарплаты с учётом НДФЛ, производственного календаря РФ и отпусков.

## Цель

Помочь пользователю рассчитать даты и суммы получения зарплаты (аванс + зарплата) на весь год, с учётом:
- Прогрессивной шкалы НДФЛ в РФ
- Производственного календаря (выходные и праздничные дни)
- Дополнительных доходов (премии)
- Отпусков (оплачиваемые и за свой счёт)

## Стек

- **React 19** + **TypeScript**
- **Vite** — сборщик
- **pnpm** — менеджер пакетов
- **TanStack Query** — управление данными
- **shadcn/ui** — UI-компоненты
- **Tailwind CSS** — стилизация
- **date-fns** — работа с датами
- **zod** — валидация

## Архитектура

Приложение работает полностью на клиенте (SPA). Бэкенда нет, данные не хранятся на сервере.

### Хранение данных

Все настройки пользователя сохраняются в `localStorage` браузера.

### Внешние API

- **isdayoff.ru** — производственный календарь РФ
  - Endpoint: `https://isdayoff.ru/api/getdata?year=YYYY&cc=ru`
  - Возвращает строку цифр по одному на каждый день года: `0`=рабочий, `1`=нерабочий, `2`=сокращённый, `8`=праздник

## Функциональные требования

### 1. Настройки зарплаты

Пользователь вводит:
- Оклад до НДФЛ (gross salary) — числовое поле в рублях
- Дату получения аванса (число месяца, напр. 10)
- Дату получения зарплаты (число месяца, напр. 25)
- Способ распределения оплаты:
  - **50/50** — аванс и зарплата по 50% от оклада
  - **По отработанным дням** — аванс пропорционален рабочим дням с 1 по 15, остаток — в зарплате

Аванс платится на `salaryDay` месяца M, зарплата — на `advanceDay` месяца M+1. В декабре — полная выплата на `salaryDay`.

### 2. Премии

Пользователь может добавить премию:
- Дата выплаты
- Тип: «В окладах» (премия = кол-во окладов) / «Своя сумма» (фиксированная gross-сумма)
- Сумма премии

Премия влияет на накопленный доход с начала года и, соответственно, на ставку НДФЛ для последующих выплат.

### 3. Отпуска

Пользователь может добавить несколько отпусков за год:
- Дата начала и окончания
- Тип: «Оплачиваемый» / «За свой счёт»

**Расчёт отпускных (по ТК РФ):**
- `СДЗ = доход12мес / 12 / 29,3` (среднемесячное число календарных дней)
- `Отпускные = СДЗ × количество календарных дней отпуска`
- Доход за 12 мес берётся из настройки «Доход за последние 12 месяцев», если не указан — `оклад × 12`

**Выплата отпускных:**
- За 2 дня до начала отпуска, с учётом выходных (смещение назад на рабочий день)
- Отпускные облагаются НДФЛ по прогрессивной шкале и влияют на накопленный доход

**Вычитание из ЗП/аванса:**
- Рабочие дни отпуска вычитаются из соответствующих выплат (`dayRate = оклад / рабочиеДниМесяца`)
- По отработанным дням: аванс считается только от рабочих дней без отпуска (1–15 минус отпускные)
- 50/50: вычитаем из аванса; если ушёл в минус — переносим избыток на зарплату

### 4. Производственный календарь

Приложение загружает данные из API isdayoff.ru:
- Праздничные дни, сокращённые рабочие дни, выходные

Если дата выплаты зарплаты попадает на выходной или праздник, дата смещается **назад** на ближайший рабочий день.

### 5. Расчёт НДФЛ (прогрессивная шкала 2026)

НДФЛ рассчитывается по прогрессивной шкале с учётом накопленного дохода с начала года:

| Накопленный доход за год | Ставка НДФЛ |
|--------------------------|-------------|
| До 2 400 000 ₽           | 13%         |
| 2 400 000 — 5 000 000 ₽  | 15%         |
| 5 000 000 — 20 000 000 ₽ | 18%         |
| 20 000 000 — 50 000 000 ₽| 20%         |
| Свыше 50 000 000 ₽       | 22%         |

Повышенная ставка применяется только к сумме превышения порога.

Для каждой выплаты рассчитывается:
- **Оклад** (gross) — сумма до вычета НДФЛ
- **НДФЛ** — налог по прогрессивной шкале
- **На руки** (net) — оклад минус НДФЛ

### 6. Календарь на год

Визуализация в виде сетки из 12 месяцев:
- Дни выплат зарплаты отмечены **зелёным** цветом
- Дни выплаты отпускных отмечены **фиолетовым** цветом
- Дни выплаты премий отмечены **синим** цветом
- Дни отпуска отмечены **серым** (скругление только по краям серии)
- Текущая дата отмечена **жёлтым** цветом
- Выходные и праздничные дни — **красный** цвет

Клик по дню выплаты открывает попап с детальной информацией. При наведении — тултип с суммой.

## Структура проекта

```
src/
├── components/        # UI-компоненты
│   ├── ui/            # shadcn/ui + DatePicker, MoneyInput
│   ├── Calendar/      # календарь на год с popover и tooltip
│   ├── Settings/      # форма настроек зарплаты
│   ├── Bonus/         # управление премиями
│   └── Vacation/      # управление отпусками
├── hooks/             # useSalaryCalc, useCalendar, useBonuses, useVacations и т.д.
├── lib/               # утилиты (cn, форматирование)
├── services/          # сервисы (localStorage, isdayoff API)
├── types/             # TypeScript-интерфейсы + Zod schemas
└── App.tsx            # главный компонент
```
