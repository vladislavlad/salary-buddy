# Salary Buddy

Клиентский калькулятор зарплаты с учётом НДФЛ, производственного календаря РФ, премий, доплат, отпусков и больничных.

## Приложение

[Salary Buddy на GitHub Pages](https://vladislavlad.github.io/salary-buddy/)

## Что делает приложение

Salary Buddy рассчитывает календарь выплат на год:

- авансы и зарплаты;
- ежемесячные доплаты;
- премии (в окладах или фиксированные);
- отпускные по ТК РФ;
- больничные: пособие от работодателя и СФР, доплата до оклада, все типы причин (ст. 7 ФЗ-255, ст. 9 ФЗ-125);
- НДФЛ по прогрессивной шкале 2026;
- перенос дат выплат с выходных и праздников на предыдущий рабочий день.

Пользователь может подтвердить или изменить фактическую gross-сумму платежа – приложение пересчитает НДФЛ для последующих выплат.

**Приложение работает как SPA без бэкенда. Данные сохраняются в `localStorage` браузера!**

## Стек

- React 19 + TypeScript + Vite
- pnpm
- TanStack Query для загрузки производственного календаря
- shadcn/ui + Tailwind CSS
- zod для валидации данных
- Vitest + Testing Library

## Команды

```bash
pnpm install      # установить зависимости
pnpm dev          # запуск dev-сервера
pnpm build        # сборка продакшена
pnpm preview      # просмотр сборки
pnpm lint         # линтинг
pnpm format       # форматирование
pnpm test         # тесты в watch-режиме
pnpm test:run     # однократный запуск тестов
```

## Экспорт / импорт данных

Все данные можно экспортировать в JSON-файл и импортировать обратно. Совместимость со старыми форматами не поддерживается.
