# AGENTS.md — Salary Buddy

## Обзор проекта

Salary Buddy — SPA для расчёта дат и сумм получения зарплаты в РФ с учётом прогрессивной шкалы НДФЛ и производственного календаря.

## Стек

- React 19 + TypeScript + Vite + pnpm
- TanStack Query (управление состоянием запросов)
- shadcn/ui + Tailwind CSS (UI)
- date-fns (работы с датами)
- zod (валидация)
- localStorage (персистентность данных)

## Ключевые правила расчёта

### НДФЛ 2026 (прогрессивная шкала РФ)

| Накопленный доход | Ставка |
|---|---|
| До 2 400 000 ₽ | 13% |
| 2.4 — 5 млн ₽ | 15% |
| 5 — 20 млн ₽ | 18% |
| 20 — 50 млн ₽ | 20% |
| Свыше 50 млн ₽ | 22% |

Ставка применяется только к сумме превышения порога. НДФЛ считается от накопленного дохода с начала года, затем вычитается уже уплаченный налог.

### Производственный календарь

- Данные загружаются из `https://raw.githubusercontent.com/isdayoff/calendars/main/db/YYYY/ruYYYY.json`
- Если дата выплаты попадает на выходной (суббота/воскресенье) или праздник — смещаем **назад** на ближайший рабочий день
- Выходные и праздники на календаре: красный цвет

### Распределение зарплаты

Два режима:
1. **50/50**: аванс = 50% оклада, зарплата = 50% оклада
2. **По отработанным дням**: аванс пропорционален количеству рабочих дней в периоде (с начала месяца до даты аванса), остаток — в зарплате

### Премии

- Пользователь добавляет премию с датой и суммой
- Премия учитывается в накопленном доходе, влияя на ставку НДФЛ для последующих выплат
- Типы: «В окладах» (gross) / «Своя сумма» (net)

## Цветовая схема календаря

- 🟢 Зелёный — день выплаты зарплаты
- 🔴 Красный — выходной или праздник
- 🔵 Синий — текущая дата

## Конвенции кода

- **Строгий TypeScript**: `strict: true`, без `any`
- **Именованные экспорты** по умолчанию, дефолтный экспорт только для компонентов страниц
- **camelCase** для переменных/функций, **PascalCase** для компонентов и типов
- **zod** для валидации пользовательского ввода
- **TanStack Query** для всех асинхронных запросов (загрузка календаря)
- Комментарии только на русском языке, если они нужны
